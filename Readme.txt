

Question Classification Using NLP

Overview
This project focuses on Question Classification using Natural Language Processing (NLP) techniques. The dataset used in this project is provided by Xin Li and Dan Roth, and is widely recognized for its contributions to question classification research. The dataset includes question class definitions, training and testing sets, preprocessing scripts, feature definition Tools.

The project leverages the TREC Question Classification Dataset and utilizes various NLP techniques for preprocessing, feature extraction, and model training.

Dataset: https://cogcomp.seas.upenn.edu/Data/QA/QC/
The dataset used in this project contains:

Question Classification Taxonomy: Defines the classes used for categorizing questions.
Training Sets:
Set 1: 1000 labeled questions
Set 2: 2000 labeled questions
Set 3: 3000 labeled questions
Set 4: 4000 labeled questions
Set 5: 5500 labeled questions
Test Set:
TREC-10 questions used to evaluate the model's performance.
